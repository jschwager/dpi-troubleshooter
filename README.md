# DreamPi Troubleshooter

A diagnostic script for [DreamPi](https://github.com/Kazade/dreampi) that identifies setup issues and troubleshoots connection problems.

## Quick Start

Run the following command on your DreamPi to download the script:

```bash
wget -O troubleshoot.sh https://raw.githubusercontent.com/jschwager/dpi-troubleshooter/main/troubleshoot.sh && chmod +x troubleshoot.sh
```

Once the script has been downloaded, it can be initialized via the following command:

```bash
./troubleshoot.sh
```

## What It Does

This script performs comprehensive diagnostics on your DreamPi installation, including:

- Configuration validation
- Connection testing
- Network diagnostics
- Service status checks
- Log analysis

## Requirements

- DreamPi device
- Internet connectivity
- Bash shell
